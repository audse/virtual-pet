# Types

These are the definitions for common types throughout all the APIs.

## Position
```ts
interface Position {
    x?: number, // default = 0
    y?: number, // default = 0
    z?: number, // default = 0
}
```

## Dimensions
```ts
interface Dimensions {
    width?: number, // default = 1
    height?: number, // default = 1
    depth?: number, // default = 1
}
```
